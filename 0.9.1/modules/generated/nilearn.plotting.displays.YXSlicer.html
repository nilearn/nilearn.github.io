8.11.31.27. nilearn.plotting.displays.YXSlicer¶

class nilearn.plotting.displays.YXSlicer(cut_coords, axes=None, black_bg=False, brain_color=(0.5, 0.5, 0.5), **kwargs)[source]¶

The YXSlicer class enables to combine coronal and sagittal views on the same figure with plotting functions of Nilearn like nilearn.plotting.plot_img. This visualization mode can be activated by setting display_mode='yx':

from nilearn.datasets import load_mni152_template
from nilearn.plotting import plot_img
img = load_mni152_template()
# display is an instance of the YXSlicer class
display = plot_img(img, display_mode='yx')
