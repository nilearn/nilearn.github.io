6.1. nisl.datasets: Automatic Dataset Fetching¶

Utilities to download NeuroImaging datasets

User guide: See the Dataset loading utilities section for further details.

6.1.1. Functions¶

datasets.fetch_haxby([data_dir, n_subjects, ...]) Download and loads complete haxby dataset
datasets.fetch_haxby_simple([data_dir, url, ...]) Download and loads an example haxby dataset
datasets.fetch_nyu_rest([n_subjects, ...]) Download and loads the NYU resting-state test-retest dataset
datasets.fetch_adhd([n_subjects, data_dir, ...]) Download and loads the ADHD resting-state dataset

6.2. nisl.utils: Manipulating Niimgs¶

Validation and conversion utilities.

User guide: See the utils section for further details.

6.2.1. Functions¶

utils.is_a_niimg(object) Check for get_data and get_affine method in an object
utils.check_niimg(niimg) Check that an object is a niimg and load it if necessary
utils.check_niimgs(niimgs[, accept_3d]) Check that an object is a list of niimg and load it if necessary
utils.concat_niimgs(niimgs) Concatenate a list of niimgs

6.3. nisl.masking: Data Masking Utilities¶

Utilities to compute a brain mask from EPI images

User guide: See the Masking the data: from 4D image to 2D array section for further details.

6.3.1. Functions¶

masking.compute_epi_mask(mean_epi[, ...]) Compute a brain mask from fMRI data in 3D or 4D ndarrays.
masking.compute_multi_epi_mask(session_epi) Compute a common mask for several sessions or subjects of fMRI data.
masking.intersect_masks(input_masks[, ...]) Compute intersection of several masks
masking.apply_mask(niimgs, mask_img[, ...]) Extract time series using specified mask
masking.unmask(X, mask[, transpose]) Take masked data and bring them back into 3D

6.4. nisl.resampling: Data Resampling Utilities¶

Utilities to resample a Nifti Image

User guide: See the Resampling section for further details.

6.4.1. Functions¶

resampling.to_matrix_vector(transform) Split a transform into it’s matrix and vector components.
resampling.from_matrix_vector(matrix, vector) Combine a matrix and vector into a homogeneous transform.
resampling.get_bounds(shape, affine) Return the world-space bounds occupied by an array given an affine.
resampling.resample_img(niimg[, ...]) Resample a Nifti Image

6.5. nisl.signals: Preprocessing Time Series¶

Preprocessing functions for time series.

User guide: See the signals section for further details.

6.5.1. Functions¶

signals.clean(signals[, confounds, ...]) Normalize the signal, and if any confounds are given, project in the orthogonal space.

6.6. nisl.io: Loading and Preprocessing files easily¶

The nisl.io module includes scikit-learn tranformers and tools to preprocess neuro-imaging data.

User guide: See the The NiftiMasker: loading, masking and filtering and nifti_masker_advanced section for further details.

6.6.1. Classes¶

nifti_masker.NiftiMasker([sessions, mask, ...]) Nifti data loader with preprocessing
nifti_multi_masker.NiftiMultiMasker([mask, ...]) Nifti data loader with preprocessing for multiple subjects
