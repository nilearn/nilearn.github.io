8.12.15.9. nilearn.glm.first_level.mean_scaling¶

nilearn.glm.first_level.mean_scaling(Y, axis=0)¶

Scaling of the data to have percent of baseline change along the specified axis

Parameters:

Y : array of shape (n_time_points, n_voxels)

The input data.

axis : Axis along which the scaling mean should be calculated. Default 0.
Returns:

Y : array of shape (n_time_points, n_voxels),

The data after mean-scaling, de-meaning and multiplication by 100.

mean : array of shape (n_voxels,)

The data mean.
