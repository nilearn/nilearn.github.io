9.2.2. Glass brain plotting in nilearn¶

See Plotting brain images for more plotting functionalities.

9.2.2.1. Retrieve data from Internet¶

from nilearn import datasets

motor_images = datasets.fetch_neurovault_motor_task()
stat_img = motor_images.images[0]

9.2.2.2. Glass brain plotting: whole brain sagittal cuts¶

from nilearn import plotting

plotting.plot_glass_brain(stat_img, threshold=3)

Out:

<nilearn.plotting.displays.OrthoProjector object at 0x7f8bfc45cfa0>

9.2.2.3. Glass brain plotting: black background¶

On a black background (option “black_bg”), and with only the x and the z view (option “display_mode”).

plotting.plot_glass_brain(
    stat_img, title='plot_glass_brain',
    black_bg=True, display_mode='xz', threshold=3)

Out:

<nilearn.plotting.displays.XZProjector object at 0x7f8bfc075d60>

9.2.2.4. Glass brain plotting: Hemispheric sagittal cuts¶

plotting.plot_glass_brain(stat_img,
                          title='plot_glass_brain with display_mode="lyrz"',
                          display_mode='lyrz', threshold=3)

plotting.show()
