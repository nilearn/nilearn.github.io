8.11.31.3. nilearn.plotting.displays.OrthoProjector¶

class nilearn.plotting.displays.OrthoProjector(cut_coords, axes=None, black_bg=False, brain_color=(0.5, 0.5, 0.5), **kwargs)[source]¶

A class to create linked axes for plotting orthogonal projections of 3D maps. This visualization mode can be activated from plot_glass_brain, by setting display_mode='ortho':

from nilearn.datasets import load_mni152_template
from nilearn.plotting import plot_glass_brain
img = load_mni152_template()
# display is an instance of the OrthoProjector class
display = plot_glass_brain(img, display_mode='ortho')

  • If not None, either or both of these values will be used to as the minimum and maximum values to color edges.

  • If it is a number only the edges with a value greater than edge_threshold will be shown.
